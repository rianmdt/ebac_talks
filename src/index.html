Front-End

Front-end é a parte visual de um site ou aplicação, que é a interface gráfica do usuário (GUI) com a qual as pessoas interagem. O front-end inclui elementos como botões, caixas de seleção, gráficos, imagens e menus de navegação.

UI/UX

UX e UI são áreas complementares e importantes no desenvolvimento de produtos digitais. A experiência do usuário é a base para o design da interface, por isso, o conhecimento em UX é fundamental para o UI.

Data-Science

Data Science é uma área que estuda a coleta, processamento, tratamento, análise, modelagem e visualização de dados. É uma abordagem multidisciplinar que combina conhecimentos de matemática, estatística, inteligência artificial e engenharia da computação.

Back-End

O back-end é a parte lógica de um site ou aplicação, que fica por trás daquilo que o usuário vê e interage. Ele é responsável por processar as solicitações do usuário, garantindo que os dados sejam salvos e processados corretamente.
